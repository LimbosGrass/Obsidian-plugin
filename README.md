# Obsidian 时间轴插件

一个为 Obsidian 设计的时间轴插件，以垂直时间轴格式显示待办事项，支持任务管理、注释和时间提醒功能。

## 功能特性

- **时间轴视图**：以垂直时间轴格式显示待办事项
- **两侧布局**：任务交替显示在时间轴的左右两侧
- **任务管理**：通过复选框标记任务完成状态
- **注释功能**：右键点击任务添加注释
- **时间提醒**：在注释中添加 `@YYYY-MM-DD HH:MM` 格式的时间，系统会在时间到达时发送通知
- **任务筛选**：支持筛选全部、已完成或未完成的任务
- **文件夹配置**：可以配置包含所有文件夹或指定特定文件夹
- **响应式设计**：适配不同屏幕尺寸

## 安装方法

1. 从 [发布页面](https://github.com/LimbosGrass/obsidian-timeline-plugin/releases) 下载最新版本
2. 将 zip 文件解压到 Obsidian 插件文件夹
   - Windows：`%APPDATA%\Obsidian\Plugins`
   - macOS：`~/Library/Application Support/Obsidian/Plugins`
   - Linux：`~/.config/obsidian/Plugins`
3. 在 Obsidian 设置中启用插件

## 使用方法

### 打开时间轴视图
- 通过命令面板运行 "打开时间轴视图" 命令
- 或点击侧边栏的小树图标

### 管理任务
- 点击任务旁的复选框标记任务为已完成/未完成
- 双击任务跳转到原始文件
- 右键点击任务添加注释

### 添加时间提醒
在任务注释中添加 `@YYYY-MM-DD HH:MM` 格式的时间，例如：
```
- 注释: 准备会议材料 @2026-03-15 17:30
```
系统会在时间到达前后5分钟内发送通知。

### 筛选任务
- 在时间轴视图顶部点击筛选按钮，可以选择显示全部、已完成或未完成的任务

### 配置插件
在 Obsidian 设置中找到 "时间轴插件设置"，可以配置：
- 是否包含所有文件夹
- 指定要包含的文件夹路径
- 默认任务筛选类型
- 时间轴名称

## 开发指南

1. 克隆仓库
2. 运行 `npm install` 安装依赖
3. 运行 `npm run dev` 启动开发服务器
4. 修改代码
5. 运行 `npm run build` 构建生产版本
